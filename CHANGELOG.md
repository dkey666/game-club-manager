# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-08

- prepared the project for public open-source publication
- renamed project metadata to `Dkx Game Club OS/Manager`
- added `README.md`, `LICENSE`, `docs/`, `ROADMAP`, and dependency manifests
- added `.env.example` and safer ignore rules for secrets, logs, and local databases
- switched email dependency loading to Composer-compatible PHPMailer setup
- reduced Node metadata to the actual local database bootstrap utility
- removed hardcoded deployment assumptions from several key entry points
